# Read Me First


### prerequisite:
java 1.8 is required to run the stream.

### Guides
1. Download the source code from git
2. Import the Project(as maven project) resource-allocator in any IDE Eclipse/Intellij
3. Run as java application on the ResourceAllocatorApplication.java  available from \resource-allocator\src\main\java\com\poshmark\resourceallocator


### Test Cases
1. By Default 4 test cases are available in ResourceAllocatorApplication.java
